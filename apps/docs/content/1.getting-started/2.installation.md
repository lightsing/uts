## CLI Installation

The UTS CLI is the primary tool for timestamping files and verifying timestamps.

### From crates.io (Recommended)

```bash
cargo install uts-cli --version 0.1.0-alpha.0 --locked
```

### From Source

```bash
git clone https://github.com/lightsing/uts
cd uts
cargo install --path crates/cli
```

### Verify Installation

```bash
uts --version
```

You should see output like:

```
uts-cli 0.1.0-alpha.0
```

## TypeScript SDK

Install the TypeScript/JavaScript SDK for browser or Node.js applications:

```bash
pnpm add @uts/sdk
```

Or with npm:

```bash
npm install @uts/sdk
```

Or with yarn:

```bash
yarn add @uts/sdk
```

### Verify Installation

```typescript
import { SDK } from '@uts/sdk'

const sdk = new SDK()
```

## Python SDK

Install the Python SDK:

```bash
pip install uts-sdk
```

Or with pipenv:

```bash
pipenv install uts-sdk
```

### Verify Installation

```python
from uts import SDK

sdk = SDK()
```

## Go SDK

Install the Go SDK:

```bash
go get github.com/uts-dot/sdk-go
```

### Verify Installation

```go
package main

import (
    "github.com/lightsing/uts/packages/sdk-go/uts"
)

func main() {
    sdk := NewSDK()
}
```
